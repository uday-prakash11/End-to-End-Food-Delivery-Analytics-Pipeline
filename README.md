# End-to-End Food Delivery Analytics Pipeline

### Project Overview

- This project analyzes a Zomato food delivery dataset to uncover insights about restaurant performance, customer behavior, and cuisine popularity. The dataset contains multiple relational tables including restaurants, orders, food items, menus, and users.

- The goal of this project is to demonstrate how raw relational data can be transformed into meaningful business insights using SQL analytics, Python visualization, and dashboard reporting.

- This analysis can help food delivery platforms and restaurant businesses optimize pricing strategies, identify high-performing restaurants, and understand customer ordering patterns.


### Project Objectives

- Analyze restaurant performance and popularity

- Identify top cuisines and trending food items

- Study customer spending behavior

- Evaluate order trends over time

- Generate actionable insights for business decision making

### Dataset Description

The dataset contains five relational tables representing a food delivery platform database.

| Table              | Description                                                  |
| ------------------ | ------------------------------------------------------------ |
| **food.csv**       | Food items and veg/non-veg classification                    |
| **menu.csv**       | Restaurant menus and food pricing                            |
| **orders.csv**     | Customer order transactions                                  |
| **restaurant.csv** | Restaurant details including location, ratings, and cuisines |
| **users.csv**      | Customer demographic information                             |


### Tools & Technologies

- PostgreSQL – Database management and advanced SQL queries

- SQL – Data extraction and business analytics

- Python – Data preprocessing and exploratory analysis

- Pandas & NumPy – Data manipulation and transformation

- Matplotlib & Seaborn – Data visualization

- Tableau – Interactive dashboards and business insights
